Bienvenidos.

Seminario Desarrollo de API's para Aplicaciones en PHP. El seminario de Desarrollo de API's para Aplicaciones en PHP presenta de manera amplia los conceptos que ha de tener una API para trabajar con las diferentes partes que la consumen hoy por hoy. También hablaremos de los diferentes esquemas de seguridad que intervienen entre dichas partes y como se implementan usando las tecnologías del mercado PHP y Zend Framework para desarrollo de APIs. Como colofón, se apuntarán las ideas de desarrollo de PHP 7 y Zend Framework 3, versiones nuevas a ser liberadas en 2015.
